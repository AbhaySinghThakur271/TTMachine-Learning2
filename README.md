# 🌍 Global Pollution Analysis and Energy Recovery

## 📌 Project Overview

This project analyzes global pollution data to understand the relationship between pollution levels and energy recovery. It applies data analysis, visualization, and machine learning techniques to generate insights and predictive models.

The goal is to explore how pollution can be reduced and efficiently converted into a valuable energy resource.

---

## 🎯 Objectives

* Analyze pollution trends across countries and years
* Perform data preprocessing and feature engineering
* Build predictive models for energy recovery
* Classify pollution levels into categories
* Provide actionable insights and recommendations

---

## 📂 Dataset

The dataset includes:

* Air, Water, and Soil Pollution Index
* CO₂ Emissions
* Industrial Waste (tons)
* Energy Consumption and Recovery (GWh)
* Country and Year

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## 🔍 Project Workflow

### 1. Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature scaling and normalization
* Cleaning column names

### 2. Exploratory Data Analysis (EDA)

* Descriptive statistics
* Correlation analysis (heatmaps)
* Data visualization (line plots, box plots, bar charts)

### 3. Feature Engineering

* Energy consumption per capita
* Year-based trend analysis

### 4. Machine Learning Models

#### 🔹 Linear Regression

* Predicts energy recovery (GWh)
* Evaluation: R² Score, MSE, MAE

#### 🔹 Logistic Regression

* Classifies pollution levels (Low, Medium, High)
* Evaluation: Accuracy, Precision, Recall, F1-score

---

## 📊 Key Findings

* Higher pollution levels are associated with higher energy recovery potential
* Industrial waste significantly contributes to energy generation
* Some regions show inefficient utilization of pollution resources
* Moderate pollution levels provide balanced optimization opportunities
